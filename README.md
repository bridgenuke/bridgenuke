### BridgeNuke Project
**Steps**
- Copy that floppy
- Boot that floppy
- Connect to network
- Configure BIOS to boot from the hard drive.
- Copy to Hard Drive
- Copy to SD Card
- Copy either to the hard drive or the SD Card.
- Boot from floppy into linux.
- Install whatever ISA/PCI cards you want!
